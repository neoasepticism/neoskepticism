웰니스의 역설

'몸에 좋다'고 배운 음식이, 사실은 세균의 먹이였습니다.

아래는 건강식으로 권장되어 왔으나, 장내 발효의 기질로 지목되는 것이다.

섬유질
= 좋은 균의 먹이
→ 난소화성 발효 기질
통곡물·잡곡
= 저속노화
→ 발효 시간 연장
발효식품
= 프로바이오틱
→ 외부 균 직접 투입
마늘·양파
= 항염·면역
→ 식물 황(黃) 공급
프로바이오틱스
= 좋은 균 보충
→ 검증 안 된 세균
생채소·날것
= 효소·비타민
→ 세균 직접 섭취
기전 · 하나의 인과 사슬

식단에서 시작해, 변연계에서 끝납니다.

장내 세균들이 만들어내는 황 가스가 두 방향으로 몸을 흔듭니다.

난소화성 식단 + 황

섬유·잡곡·발효식품·마늘이 소장에서 다 흡수되지 못하고 발효의 재료가 됩니다.

▼

SRB·후소박테리움 과증식

황과 난소화성 기질이 만나면 황 가스를 생성하는 세균이 증식합니다.

▼

황 가스 생산 — H₂S · 메틸메르캅탄

입 냄새, 체취, 독한 방귀 냄새로 나타납니다. 황화수소와 메틸메르캅탄(CH₃SH)은 시안화물(청산가리)와 같은 계열의 미토콘드리아 독성을 가집니다.

▼

① GLP-1 억제 → 변연계 폭주

SRB가 포만 호르몬 GLP-1의 분비를 억제 → 식욕 조절 실패, 도파민 과잉, 중독성 충동, 자기 파괴적 행동으로 이어집니다.

▼

② 가스 재흡입 → 뇌·미토콘드리아

매일 자기가 만든 가스를 다시 들이마시면 변연계가 손상됩니다 — 불안, 강박, 브레인포그, 만성 피로. 같은 균(후소박테리움)이 불안장애와 대장암에도 관여합니다.
